# Git Client for GitHub — Feedback

This is the public feedback tracker for **Git Client for GitHub**, a native iOS app for browsing repositories, reviewing changes, and syncing files between GitHub and your device.

→ [Download on the App Store](#https://apps.apple.com/de/app/git-client/id6758091553)

---

## How to submit feedback

**Bug report** — something isn't working as expected
**Feature request** — something you'd like to see added

Use the [Issues tab](../../issues) above and pick the matching template.

Please search existing issues before opening a new one — your request may already be tracked.

---

## What this repo is for

This repo exists solely to collect feedback from users. The app's source code is not hosted here.
